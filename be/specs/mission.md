# 백엔드 미션

Team Manager 백엔드는 축구팀 운영에 필요한 핵심 업무 규칙을 안정적으로
보존하고, 프론트엔드와 외부 클라이언트가 예측 가능한 API로 사용할 수
있는 서비스를 제공한다.

## 우리가 해결하는 문제

축구팀 운영에는 팀 생성, 팀원 관리, 역할 관리, 경기 일정, 경기 기록,
참가자 관리, 게스트 선수, 회비 관리처럼 서로 연결된 규칙이 많다.
백엔드는 이 규칙을 코드에 흩어두지 않고 명세, 도메인 모델, API 계약,
테스트로 추적 가능하게 만든다.

## 주요 사용자

- 팀 운영자: 팀 정보, 팀원, 경기, 회비를 관리한다.
- 팀원: 경기 일정과 참여 정보를 확인하고 필요한 작업을 수행한다.
- 프론트엔드 클라이언트: 문서화된 API 계약을 기준으로 화면을 구현한다.

## 성공 기준

- 핵심 업무 규칙이 명세와 테스트로 설명된다.
- API 응답과 에러 형식이 일관된다.
- 권한이 필요한 작업은 명확히 보호된다.
- 기능 변경 시 기존 클라이언트 영향이 드러난다.
