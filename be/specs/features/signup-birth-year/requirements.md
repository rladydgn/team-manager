# 기능 요구사항

## 기능 이름

회원가입 출생연도 입력

## 목표

회원가입 시 태어난 연도를 필수로 받고, 1월 1일로 고정한 `LocalDate`로 계정 정보에 저장한다.

## 인수 기준

- 회원가입 요청은 `birthDate` 날짜 값을 필수로 포함한다.
- 출생연도는 1900년부터 현재 연도까지 허용한다.
- `birthDate`의 월과 일은 반드시 1월 1일이어야 한다.
- 범위를 벗어나거나 1월 1일이 아닌 값은 `INVALID_REGISTER_REQUEST`로 거절한다.
- 저장된 값은 `users.birth_date`의 `DATE` 컬럼에 보존한다.
