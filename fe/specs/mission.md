# 프론트엔드 미션

Team Manager 프론트엔드는 축구팀 운영자가 팀, 팀원, 경기, 참가, 회비
흐름을 빠르게 이해하고 처리할 수 있는 명확한 사용자 경험을 제공한다.

## 우리가 해결하는 문제

팀 운영 업무는 반복적이고 상태가 자주 바뀐다. 프론트엔드는 사용자가
현재 상태를 빠르게 파악하고, 필요한 조치를 실수 없이 수행하도록
돕는다.

## 주요 사용자

- 팀 운영자: 팀과 경기 운영 정보를 관리한다.
- 팀원: 일정, 참가, 기록, 공지성 정보를 확인한다.
- 신규 사용자: 가입과 로그인 과정을 거쳐 팀에 참여한다.

## 성공 기준

- 주요 업무 흐름이 적은 이동으로 완료된다.
- API 실패와 권한 문제를 사용자가 이해할 수 있다.
- 모바일과 데스크톱에서 핵심 화면이 안정적으로 보인다.
- 화면 구현이 명세와 API 계약을 추적할 수 있다.
