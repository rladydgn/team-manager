# 기능 요구사항

## 기능 이름

팀 가입 신청 관리

## 목표

사용자는 팀 가입을 신청하고, 운영진은 별도 화면에서 신청자를 승인 또는 거부한다.

## 인수 기준

- 팀 목록과 상세의 가입 동작은 신청 완료 안내를 표시한다.
- 팀 상세에서 OWNER와 SUB_MANAGER는 가입 신청 관리 화면으로 이동할 수 있다.
- 관리 화면은 대기 중인 신청자 이름과 신청 시각을 표시한다.
- 승인과 거부 동작은 각 신청 행에서 가능하며 처리 결과가 즉시 반영된다.
- 권한이 없는 사용자는 관리 화면에서 신청 목록을 볼 수 없다.
